Trader Performance vs Market Sentiment
This project analyzes how market sentiment (Fear vs Greed) affects trader behavior and performance on the Hyperliquid platform.

Contents
Data preparation and daily trader-level aggregation Performance and behavioral analysis across sentiment regimes Trader segmentation based on trading frequency Simple predictive model for trader profitability Behavioral clustering using KMeans Lightweight Streamlit dashboard for interactive exploration
How to Run
 pip install -r requirements.txt jupyter notebook streamlit
streamlit run dashboard.py
